# ExpressJS

### What and Why?

- Server Logic is Complex and you want to focus on business logic, isntead of the nitty gritty details
- ExpressJS is a framework that does the heavy lifting
- Framework: Helper functions, tools & rules that help you build your application.
- Alternatives to Express.js:
  - Vanilla NodeJS
  - Adonis.js
  - Koa
  - Sails.js
- Express.js is all about middleware
