# Introduction

### What is NodeJS?

- NodeJS is a JavaScript Runtime.
- Built on JS so nodejs allows you to run JS outside of the browser
- NodeJS uses V8 (written in C++) the JavaScript Engine built by GOOGLE
- An engine takes JS code and compiles it to machine code
- You can run JS scripts in your computer using NodeJS

### Understanding NodeJS usage:

- User Web browser which then makes a request to the server that returns HTML content
- Said Server is responsible for:
  - Database
  - Authentication
  - Input Validation
  - Your Business Logic

> NodeJS is not limited to the Server or Browser. You can use it for Utility Scripts, Build Tools, etc...

With NodeJS:

- You create a Server and Listen to Incoming Requests
- Handle Requests , Valdiate Input, Connect to DB
- Return responses (Rendered HTML, JSON, etc...)

The REPL:

- Read: Read User Input
- Eval: Evaluate User Input
- Print: Print Output
- Loop: Result
